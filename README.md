# Kubernetes Cluster AWS Terraform

## Commands

- terraform init
- terraform plan
- terraform apply
- terraform apply --auto-approve
- terraform destroy

- cp kubeconfig ~/.kube/config
- kubectl get nodes
- kubectl get svc
- kubectl get all
- kubectl create deploy nginx --image=nginx
- kubectl get pods
- kubectl port-forward pod/pod-name 8181:80
- www.localhost:8181
